# 🌎 Tourism Cinematic Landing

A cinematic tourism landing page built to create an immersive visual experience through smooth scrolling, stacked section transitions, and advanced animation techniques.

The goal of this project is to combine modern frontend architecture with high-quality motion design, creating a storytelling experience where each section feels like a new scene.

The landing experience includes scroll-driven animations, smooth transitions, reusable components, and an interactive section indicator system.

## ✨ Overview

This project was created to explore advanced frontend animation patterns and build a premium landing page experience inspired by modern travel and luxury websites.

The main focus is not only visual design, but also implementing professional frontend practices:

- Scalable component architecture
- Feature-based project organization
- Reusable UI components
- Custom hooks for complex logic
- Advanced scroll animations
- Smooth user interactions

The experience is built around a cinematic stacked scroll system where sections transition vertically while maintaining a continuous storytelling flow.

# 🚀 Features

## 🎞️ Cinematic Scroll Experience

The landing page uses a stacked section system where each scene transitions smoothly into the next.

Features:

* Full-screen sections
* Scroll-controlled animations
* Section stacking transitions
* Pin-based scrolling experience
* Smooth scene changes

---

## 📍 Interactive Section Indicator

A custom navigation indicator synchronized with the scroll position.

Includes:

* Current section tracking
* Upcoming section preview
* Dynamic progress line
* Animated section changes
* Active section highlighting

The indicator state is controlled globally through Zustand and updated from GSAP ScrollTrigger progress.

---

## 🎬 Advanced Animation System

Animations are handled through reusable GSAP hooks.

Implemented:

* Section entrance animations
* Scroll-triggered transitions
* Timeline based animations
* Text and number reveals
* Smooth state transitions

The animation logic is separated from UI components to keep the architecture clean and maintainable.
